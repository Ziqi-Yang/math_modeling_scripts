5. Optimize.curve_fit() 几乎可以拟合任何函数形式，只能计算一个因变量

用非线性最小二乘进行拟合，这个函数就比较厉害了，但是速度也最慢（一般情况下）

url:

https://docs.scipy.org/doc/scipy/reference/generated/scipy.optimize.curve_fit.html?highlight=optimize%20curve_fit#scipy.optimize.curve_fit
from scipy.optimize import curve_fit
def fun(X,coef_a,coef_b,coef_c,intercept): #定义拟合的形式,第一个参数为自变量，后面是要拟合的参数
    a=X.iloc[:,0]
    b=X.iloc[:,1]
    c=X.iloc[:,2]
    return coef_a*a+coef_b*b+coef_c*c+intercept
res=curve_fit(fun,data_x,data_y,method="lm") #p0 设置初始值, maxfev=10000 设置最大迭代次数
